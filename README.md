# Join
SQL<br>
## Join란?
두 개 이상의 테이블을 서로 연결하여 데이터를 검색할 때 사용하는 방법<br>
두 개의 테이블을 마치 하나의 테이블인 것처럼 보여준다.<br>

## 기본구조
select 테이블.컬럼, 테이블.컬럼<br>
from 테이블1, 테이블2<br>
where 조건<br>

## Join문
###  급여가 3000이상인 사원번호, LAST_NAME, SALARY, 부서명, 도시명을 조회하시오.
